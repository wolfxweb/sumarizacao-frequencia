Para executar o projeto crie o anbiente virtual depois instale as dependencias do arquivo requirements.txt no seu ambiente 

Para cria o abiente no linux ou mac utilize o comando 
python3 -m venv venv

Iniciando o ambiente
source venv/bin/activate

Comando executar o requirements
pip install -r requirements.txt

Se precisar de algum pacote que não esteja listado no arquivo requirements.txt, recomendo que adicione o nome do pacote no arquivo e execute novamente o comando de instalação do requirements.
Isso garantirá que todas as dependências necessárias estejam instaladas corretamente em seu ambiente.
